lsg-quickstart

This guide will:

  • Give a designer-friendly explanation of what DocumentJS does
  • Help you install DocumentJS
  • Help you configure a Live Style Guide site
  • Explain how to use tags to write your Live Style Guide

You should start elsewhere if:

  • You only care about JavaScript documentation
  • You already use DocumentJS and want to add a Live Style Guide
  • You want to set up DocumentJS for API documentation, then add a Live Style Guide
  • You are totally lost
