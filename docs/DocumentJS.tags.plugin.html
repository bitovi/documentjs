@plugin

  • constructor
documentjs.tags.plugin  
  • source

Specifies the parent [documentjs/DocObject DocObject]'s name. The current DocObject will be displayed under the the parent in the navigation.

@plugin NAME

 * @page can.Construct.super
 * @parent can.Construct.plugins

 *|
