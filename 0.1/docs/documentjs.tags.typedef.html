@typedef

  • constructor
documentjs.tags.typedef  
  • source

Declares an alias for a more complex type. That alias can be used in [documentjs/type TYPE] declarations.

@typedef {TYPE} NAME [TITLE]

Parameters

  1. TYPE {documentjs/type}Optional

    A [documentjs/type type expression]. This is typically an object specified like: {{}}.

  2. NAME {String}

    The name of the type.

  3. TITLE {String}

    The title of the type used for display purposes.
